인텔리제이
-1) 메인과 동일, 하위에 코드작성
-2) 클래스>동시에 패키지 생성>


단축키
라인복사 : Ctrl + d
라인삭제 : Ctrl + y

🌱 Spring / JSP 개발 필수 단축키
🔹 실행 & 서버 관련

Shift + F10 → 최근 실행한 서버/프로그램 실행
Shift + F9 → 디버깅 실행
Ctrl + F2 → 실행 중지 (Tomcat 중지할 때 자주 사용)

🔹 JSP/HTML 코드 편집
Ctrl + Space → JSP/HTML 태그 자동완성 (<c:forEach>, <input> 등)
Ctrl + / → 한 줄 주석 토글 (//)
Ctrl + Shift + / → 블록 주석 토글 (/* */ or <!-- -->)
Ctrl + Alt + L → 코드 자동 정렬 (HTML/JSP 깔끔하게 정렬)
Ctrl + Alt + O → import 최적화 (JSP 상단의 Java import 정리)
Ctrl + D → 현재 줄 복사 (폼 요소 복제할 때 유용)
Ctrl + Y → 현재 줄 삭제

🔹 컨트롤러/서비스/뷰 탐색
Ctrl + 클릭 (메서드/클래스명) → 선언부로 이동
(JSP에서 ${controllerMethod} → Controller 메서드로 점프 가능)
Ctrl + B → 선언부 이동 (위와 동일, 단축키 전용)
Ctrl + Alt + B → 구현체로 바로 이동 (인터페이스 → 구현 클래스)
Alt + F7 → 해당 메서드/변수 사용처 검색 (Controller 호출 확인)

🔹 파일/리소스 검색
Ctrl + N → 클래스 검색 (Controller, Service 클래스 찾기)
Ctrl + Shift + N → 파일 검색 (JSP, HTML, CSS, JS 빠르게 열기)
Ctrl + Shift + Alt + N → 메서드/심볼 검색 (함수 이름 검색)
Shift + Shift (두 번) → 전체 검색 (클래스, JSP, 설정파일 등 전부)

🔹 디버깅 (Spring Controller/JSP 요청 확인 시)
Ctrl + F8 → 브레이크포인트 설정/해제
F7 → Step Into (메서드 안으로 들어가기)
F8 → Step Over (다음 줄 실행)
Shift + F8 → Step Out (메서드 밖으로 나오기)
Alt + F9 → 커서 위치까지 실행

🔹 리팩토링 (DAO/Service/Controller 수정 시)
Shift + F6 → 이름 변경 (변수, 메서드, 클래스명 한번에 리팩토링)
Ctrl + Alt + M → 메서드 추출
Ctrl + Alt + V → 변수 추출

👉 정리하면, Spring/JSP에서 가장 자주 쓰는 조합은:
Ctrl + B (선언부 이동),
Ctrl + Shift + N (JSP/설정 파일 찾기),
Alt + F7 (사용처 추적),
Ctrl + Alt + L (코드 정렬),
Shift + F10 (서버 실행),
Ctrl + F8 / F7 / F8 (디버깅).
